# AutoCare-Pro---Smart-Vehicle-Service-Management
Industry: Automotive
Built With: Service Cloud, Field Service, Work Orders
Problem Statement:
An auto repair shop manages service appointments through phone calls and paper forms. They lose track of vehicle service history, parts inventory, and customer follow-ups. Mechanics need mobile access to work orders, and customers want service status updates.
Salesforce Implementation:

Service Cloud: Cases for service requests
Work Orders: Service job management
Custom Objects: Vehicle__c, Service_History__c, Parts_Inventory__c
Mobile App: Salesforce Mobile for technicians
Process Builder: Service completion notifications to customers
Custom Notifications: Parts low-stock alerts
Reports: Technician productivity, customer satisfaction scores
